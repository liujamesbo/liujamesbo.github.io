快速构建自己的博客

Posted on 2024-05-21 Edited on 2024-05-27

通过hexo和Github搭建自己免费的博客，hexo框架依赖Nodejs，所以需要安装的工具包括hexo、Nodejs、git

工具安装

  1. 安装Git

    通过homebrew安装

  2. 安装node.js

    查看node和npm版本（npm是node.js的包管理器）


    如果提示command not found:node说明没有安装node，通过homebrew安装node即可，安装完成执行上文查看版本指令验证

  3. 安装hexo

    查看hexo版本hexo -v
    如果提示command not found: hexo说明没有安装hexo，通过brew安装hexo

创建博客

  1. 初始化hexo项目

    创建hexo项目

    在source/_posts文件夹下创建md文档，执行以下指令启动本地服务查看博客（http://localhost:4000）

  2. Github创建工程

    新建仓库，仓库名称必须是xxxxx.github.io格式，选择public公开

  3. 将hexo项目部署到Github Pages

    配置hexo以及主题，此处用到的是next主题，先执行主题下载指令，然后打开_config.yml，修改配置文件


    安装部署插件

    生成并部署博客

  4. 创建个人访问令牌进行身份验证

    第三步部署hexo到Github时需要输入Github用户名和密钥（访问令牌token）
    生成个人访问令牌：点击用户头像 -> Settings -> Developer Settings -> Personal access tokens -> Generate new token，将生成的token拷贝并保存到本地这是唯一一次获取，当然可以删除重新生成
    部署hexo输入用户名和密码

  5. 设置并访问Github Pages

    设置Github Pages：进入仓库点击Settings -> Pages -> Branch，选择对应分支以及文件目录save，生成对应访问地址https://xxxxx.github.io/

    配置文件中设置对应链接，重新部署，把生成的地址配置到_config.yml中（permalink）

    访问Github Pages即可看到对应的博客内容
